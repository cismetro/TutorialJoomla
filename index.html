1. Introdução

Sistema de gerenciamento de conteúdo (CMS) de código aberto, amplamente utilizado para criar e gerenciar websites. Ele oferece uma interface intuitiva e é altamente extensível, permitindo a adição de componentes, plugins e módulos. Nesta apostila, abordaremos os conceitos básicos para começar a usar o Joomla, incluindo a criação de usuários, a instalação de componentes e plugins, e como administrar o conteúdo do seu site.

2. Usuários

Você pode gerenciar usuários com diferentes níveis de acesso e permissões. Existem cinco grupos principais de usuários: Super Administrador, Administrador, Gerente, Autor e Assinante. Abaixo, explicamos como gerenciar esses usuários e atribuir permissões para cada grupo.

Para adicionar um novo usuário, vá até o menu 'Usuários' > 'Gerenciar'. Lá, você poderá adicionar novos usuários e definir suas permissões. Para gerenciar permissões, basta acessar 'Usuários' > 'Grupos de Usuários' e configurar as permissões de cada grupo.

Usuários - Grupos

Grupos de usuários são uma maneira de organizar e gerenciar os níveis de acesso e permissões dos usuários. Cada grupo de usuários pode ter permissões específicas sobre o que pode ser acessado ou modificado dentro do site. Esses grupos ajudam a controlar quem pode editar, criar ou visualizar determinado conteúdo.

Como Gerenciar Grupos de Usuários?

  1. Acessando o Gerenciador de Grupos de Usuários
    • No painel administrativo, vá para Usuários > Grupos.
    • Aqui você verá uma lista de todos os grupos de usuários já criados no site.
  2. Criando um Novo Grupo de Usuários
    • Clique em Novo no canto superior esquerdo.
    • Defina o Nome do grupo e a Descrição (opcional).
    • Se desejar, você pode criar um grupo filho de outro grupo. Por exemplo, você pode criar um grupo "Editor Júnior" dentro de um grupo principal "Editor".
    • Após preencher as informações, clique em Salvar & Fechar.
  3. Editando um Grupo de Usuários
    • Para editar um grupo de usuários existente, clique sobre o nome do grupo na lista.
    • Altere as configurações conforme necessário, como adicionar ou remover permissões, e clique em Salvar & Fechar.
  4. Excluindo um Grupo de Usuários
    • Para excluir um grupo, selecione o grupo na lista e clique em Excluir.
    • Confirme a exclusão.

Permissões e Atribuições de Grupos

Cada grupo de usuários pode ter permissões específicas que determinam suas ações no site. O Joomla permite controlar permissões de forma detalhada:

Tipos de Permissões
  • Visualizar: Permite que o grupo visualize o conteúdo.
  • Editar: Permite que o grupo edite artigos e outros conteúdos.
  • Publicar: Permite que o grupo publique conteúdo no site.
  • Excluir: Permite que o grupo exclua artigos e outros itens.
  • Gerenciar: Permite que o grupo gerencie outros usuários ou configurações administrativas.

3. Componentes

Componentes são as partes principais de um site Joomla. Eles são responsáveis por fornecer a funcionalidade básica do site. O Joomla já vem com alguns componentes instalados, como o 'Artigos', 'Menus', 'Contatos', entre outros.

Para instalar novos componentes, vá até 'Extensões' > 'Gerenciar' > 'Instalar'. Uma vez instalado, você pode configurar o componente a partir do menu 'Componentes'.

O que são Componentes

Os componentes são as extensões que fornecem funcionalidades principais ao site. Cada componente tem uma interface administrativa e uma parte que é exibida no frontend do site. Eles são como os "núcleos" de um site Joomla, controlando a maioria das operações essenciais.

Estrutura de Componentes

Cada componente geralmente consiste em três partes principais:

  1. Administração: Parte usada para configurar o componente, incluindo opções de backend.
  2. Frontend: Parte visível aos visitantes do site, onde o componente exibe conteúdo ou funcionalidades.
  3. Banco de Dados: Alguns componentes interagem com o banco de dados do site para armazenar informações, como artigos ou registros de formulários.

Exemplos de Componentes

  1. Artigos (com_content)
    • O componente padrão para exibir artigos .
    • Ele permite que você crie, edite e organize artigos e categorias.
    • O conteúdo pode ser exibido de várias formas, como artigos individuais, blogs, listas de categorias, etc.
  2. Menu
    • Gerencia a estrutura de navegação do seu site.
    • Você pode adicionar, remover e organizar os itens de menu através deste componente.
  3. Contatos (com_contact)
    • Permite criar formulários de contato para que os visitantes possam enviar mensagens diretamente ao administrador do site.

4. Plugins

Os plugins são extensões que adicionam funcionalidades específicas ao Joomla. Eles são ativados quando um determinado evento ocorre no sistema, como o envio de um formulário ou a visualização de uma página.

Para gerenciar plugins, vá em 'Extensões' > 'Plugins'. Você pode ativar, desativar e configurar plugins a partir desse menu.

5. Módulos

Os módulos são unidades de conteúdo que podem ser exibidas em diferentes posições no seu site. Os módulos podem ser menus, pesquisas, banners, entre outros.

Para gerenciar módulos, vá em 'Extensões' > 'Módulos'. Lá, você pode criar, editar e atribuir módulos a diferentes posições dentro do seu template.

Módulos

Os módulos são pequenos blocos de conteúdo que podem ser exibidos em diferentes partes do site. Eles são usados para mostrar informações como menus, pesquisas, artigos em destaque, banners e muito mais.

Acessando o Gerenciador de Módulos

  • No painel administrativo, vá até "Extensões" → "Módulos".
  • Aqui, você verá a lista de módulos ativos no site.

Criando um Novo Módulo

  1. Clique em "Novo" para adicionar um novo módulo.
  2. Escolha o tipo de módulo desejado (ex: Menu, Artigos Populares, Login, Pesquisa).
  3. Configure os seguintes campos:
    • Título: Nome do módulo.
    • Posição: Escolha onde o módulo será exibido (dependendo do template).
    • Status: Mantenha "Publicado" para ativá-lo.
    • Atribuição de Menu: Escolha em quais páginas o módulo aparecerá.
  4. Clique em "Salvar & Fechar".

Principais Tipos de Módulos

  • Menu: Exibe um menu de navegação.
  • Artigos em Destaque: Mostra artigos marcados como destaque.
  • Pesquisa: Adiciona um campo de busca ao site.
  • Login: Permite login de usuários no site.
  • HTML Personalizado: Permite adicionar conteúdo customizado (texto, imagens, código HTML).
  • Banners: Exibe anúncios ou imagens promocionais.

6. Templating e Design

O Joomla utiliza templates para controlar o design e a aparência do seu site. Existem templates padrão, mas você pode instalar novos templates para personalizar o visual do seu site.

Para gerenciar templates, vá em 'Extensões' > 'Templates'. Aqui, você pode escolher o template padrão e personalizar suas configurações.

7. Gerenciamento de Conteúdo

Para criar e gerenciar o conteúdo do seu site, o Joomla oferece a ferramenta 'Artigos'. Você pode criar artigos, categorias e gerenciar o conteúdo do seu site com facilidade.

Para adicionar artigos, vá em 'Conteúdo' > 'Artigos' > 'Adicionar Novo'. Você pode adicionar texto, imagens, vídeos e outros elementos ao seu artigo.

Além disso, você pode organizar seus artigos em categorias, facilitando a navegação do usuário no site.

Artigos

Os artigos são os principais elementos de conteúdo do site. Eles podem conter textos, imagens, vídeos e outros elementos, organizados por categorias para facilitar a navegação e o gerenciamento.

Criando um Artigo

  1. Acessar o Gerenciador de Artigos:
    • No painel administrativo, vá até "Conteúdo" → "Artigos".
    • Clique em "Novo" para criar um artigo.
  2. Preencher as Informações do Artigo:
    • Título: Nome do artigo.
    • Alias: Nome amigável para a URL (opcional, gerado automaticamente se não preenchido).
    • Categoria: Escolha uma categoria para organizar o artigo.
    • Conteúdo: Use o editor de texto para escrever e formatar o artigo.
    • Imagens e Mídia: Insira imagens e vídeos conforme necessário.
    • Tags: Adicione palavras-chave para facilitar a busca.
  3. Configurações Adicionais:
    • Status: Publicado, não publicado, arquivado ou movido para a lixeira.
    • Acesso: Defina quem pode visualizar o artigo (público, registrado, etc.).
    • Destaque: Marque como "Sim" para exibir o artigo na página inicial.
  4. Salvar e Publicar:
    • Clique em "Salvar" para manter as alterações.
    • Clique em "Salvar & Fechar" para finalizar a edição.

Artigos em Destaque

Os artigos em destaque são artigos selecionados para aparecer na página inicial ou em seções especiais do site. Eles permitem destacar conteúdos importantes e são uma forma eficaz de organizar a apresentação de informações.

Como Destacar um Artigo
  1. Criar ou Editar um Artigo
    • No painel administrativo, vá até "Conteúdo" → "Artigos".
    • Clique em "Novo" para criar um artigo ou edite um artigo existente.
  2. Marcar como Destaque
    • No campo "Destaque", selecione "Sim".
    • Clique em "Salvar & Fechar" para aplicar as alterações.

8. Segurança e Manutenção

A segurança é uma parte essencial ao usar o Joomla. Para garantir que seu site esteja protegido, é importante manter o Joomla e suas extensões sempre atualizados.

Além disso, você pode usar o menu 'Sistema' > 'Configuração Global' para configurar as opções de segurança, como permitir ou bloquear o acesso a determinadas áreas do site e configurar as permissões de usuários.

Para manter o seu site Joomla seguro, é importante também realizar backups regulares e monitorar possíveis vulnerabilidades.

Atualizações

A área de Atualizações permite que você mantenha o seu site e suas extensões atualizados, garantindo a segurança, o desempenho e a compatibilidade do sistema. A atualização do Joomla e das extensões é uma parte crucial para o bom funcionamento do seu site, já que novas versões frequentemente corrigem erros, melhoram funcionalidades e, mais importante, corrigem vulnerabilidades de segurança.

Como Acessar a Área de Atualizações

  1. Acesse o painel de administração
  2. No menu superior, clique em Sistema.
  3. No submenu, você encontrará uma opção chamada Atualizações. Clique nela para acessar a interface de atualização.

Backup

Realizar backup do seu site é uma das tarefas mais importantes para garantir que você possa restaurá-lo em caso de problemas. O backup deve incluir tanto os arquivos do site quanto o banco de dados, pois ambos são essenciais para a operação do seu site.

Como realizar um Backup

1. Backup Manual (Arquivos e Banco de Dados)

Backup dos Arquivos:

  1. Acesse o servidor onde o seu site Joomla está hospedado usando um cliente FTP (como FileZilla) ou o Gerenciador de Arquivos no painel de controle do seu servidor (como cPanel).
  2. Baixe todos os arquivos do seu site. Isso inclui a pasta principal do Joomla e todas as suas subpastas, como:
    • /images/ (imagens)
    • /media/ (mídia)
    • /templates/ (templates personalizados)
    • /components/ (componentes)
    • /modules/ (módulos)
    • /plugins/ (plugins)
    • Todos os arquivos e pastas do Joomla que estão no diretório raiz.
